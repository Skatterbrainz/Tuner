# Install-TunerBGInfo

## SYNOPSIS
Install and enable Sysinternals BGINFO

## SYNTAX

```
Install-TunerBGInfo [[-ConfigFile] <String>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Install and enable Sysinternals BGINFO

## EXAMPLES

### EXAMPLE 1
```
Install-TunerBGInfo
```

### EXAMPLE 2
```
Install-TunerBGInfo -ConfigFile "c:\windows\system32\contoso_servers.bgi"
```
